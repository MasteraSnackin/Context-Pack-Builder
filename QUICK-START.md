## Step 1: Get API Keys (15 minutes)

### Supabase (Database)
1. Go to https://supabase.com/dashboard
2. Create a new project (free tier is fine)
3. Go to **Settings** ‚Üí **API**
4. Copy:
   - `URL` ‚Üí SUPABASE_URL
   - `service_role` key ‚Üí SUPABASE_SERVICE_ROLE_KEY

### Clerk (Authentication)
1. Go to https://dashboard.clerk.com
2. Create a new application
3. Go to **API Keys**
4. Copy:
   - `Secret Key` ‚Üí CLERK_SECRET_KEY
   - `Publishable Key` ‚Üí CLERK_PUBLISHABLE_KEY

### Anthropic (LLM)
1. Go to https://console.anthropic.com/settings/keys
2. Create a new API key
3. Copy: `sk-ant-xxxxx` ‚Üí ANTHROPIC_API_KEY

## Step 2: Configure Environment (2 minutes)

Edit `.env` file in `hacknight-repo/` folder:

```env

SUPABASE_URL=https://YOUR-PROJECT-ID.supabase.co
SUPABASE_SERVICE_ROLE_KEY=YOUR-SERVICE-ROLE-KEY

# Clerk
CLERK_SECRET_KEY=sk_test_YOUR-SECRET-KEY
CLERK_PUBLISHABLE_KEY=pk_test_YOUR-PUBLISHABLE-KEY

# Anthropic
ANTHROPIC_API_KEY=sk-ant-YOUR-API-KEY
```

## Step 3: Setup Database (3 minutes)

```bash
cd hacknight-repo

# Link to your Supabase project
pnpm supabase link --project-ref YOUR-PROJECT-REF

# Push the migration (creates context_packs table)
pnpm supabase db push
```

If you don't want to link, you can manually run the migration:
1. Open Supabase dashboard ‚Üí SQL Editor
2. Copy contents of `supabase/migrations/20260220000000_create_context_packs_table.sql`
3. Run it

## Troubleshooting

### "Error: Invalid API key"
- Check `.env` file has correct keys
- Restart dev server after changing `.env`

### "No resources found"
- Make sure you're in the `hacknight-repo` directory
- Filesystem MCP searches from `process.cwd()`
- Try absolute path in demo

### "Git MCP failed"
- This is OK! Git MCP is optional
- It will silently fail and use filesystem only
- Demo-repo has git initialized, so it should work

### "Database error"
- Check Supabase project is active
- Verify migration ran successfully
- Check RLS policies (should be enabled)

### "Claude API timeout"
- LLM call can take 5-10 seconds
- This is normal!
- Loading spinner should appear
